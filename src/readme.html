3DViewer V1.0

3DViewer - это приложение для просмотра 3D моделей с функциями записи GIF, JPEG и BMP.

Управление

Программа предоставляет возможность загружать файлы формата .obj для просмотра. Можно вращать модель вдоль трех осей xyz, также можно передвигать модель по этим осям и изменять расстояние до модели. В программее можно менять цвет фона. С помошью кнопки Capture Image можно сделать снимок отрендеренной модели, а при нажатии кнопки ГИФ начнется запись.

Установка и удаление

Установка приложения на рабочий стол происходит с помощью команды make install. Удалить приложение с рабочего стола можно с помощью команды make uninstall
